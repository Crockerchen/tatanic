# 1 考核任务

Kaggle上的Titanic项目，预测生还率。

- 使用 pytorch

# 2 实现过程
  首先主要是通过matplotlib.pyplot对测试集中的列表属性与生还率之间的关系，得出了以下几点结果：

- 女性比男性的生还率高。
- 仓位等级越高生还率高。
- 有兄弟姐妹的生还率高。
- 有父母子女的生还率高。
- 船票的费用越高生还率越高。

其他有些属性由于缺失过多，不加以分析。

在建立神经网络前，先进行对数据集的取舍以及补齐。
之后将训练集总的891个分为800+91前800个拿来训练后面91个用来检验。


# 3 实现结果

通过 kaggle 的 Titanic 中给定的测试集进行训练.
最佳结果为 0.72966.
